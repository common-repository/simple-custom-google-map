The simplest ever customizable Google Map. Edit colours, markers and labels. Enter latitude and Longitude or simply drag marker into place.

== Description ==

Simple custom Google Map adds a simple, intuitive interface to create and style a Google Map to suit your website's styling. Simple drag and drop interface means that you can drag the pointer to any position on the admin preview map, and this will be mirrored on your website front-end.

== Installation ==

1. Upload the plugin files to your plugins directory, or install the plugin via WordPress.
2. Activate the plugin through the 'Plugins' screen.
3. Use the 'Map Styles' screen to configure the plugin
4. Configure the map to your heart's content, use built in widget, or add '[styled_google_map]' shortcode.
